## Basic Keys

| Keys      | Description |
| :----:        |    :----:   |
|  Cmd + z | Revert Back | 
|  Cmd + Shift + z  | Revert Back Forward | 

## IntelliJ

### Debug

| Keys      | Description |
| :----:        |    :----:   | 
| Option+F8  | Evaluate expression | 
| Option + Ctrl + H  | See call hierarchy | 
| Cmd + F  | Find | 
| Cmd + R  | Replace |
